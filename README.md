# Manga Scraper Gui

## Supported Sites
- www.mangaeden.com
- www.mangareader.net


## Install dependencies through Virtualenv (Anaconda) 

```
$ conda create -n scraper python=3.6
$ conda install -n scraper scrapy PyQt
$ source activate scraper
(scraper) $ pip install img2pdf
```

## Start the Gui
```
$ source activate scraper
(scraper) $ python app.py
```
